这里主要讲一下继电器程序的控制逻辑

数艺电子小程序里控制界面一共有24个按钮,分别是

按钮1~16,对应的按下对应的程序,main.py文件里会收到字符1~16,的16个字符串
按键a~h,对应的按下对应的程序,main.py文件里会收到字符a~h,的8个字符串

### 继电器控制方式:

先看代码:

bfCallback(dat)函数会收到巴法云转发过来的对应按键字符串,下边的代码,主要功能是解析这些字符串,并执行对应的程序

其中:
字符a,接收到表示接下边收到的1~16为控制继电器打开,
字符b,接收到表示接下边收到的1~16为控制继电器关闭,
字符c,接收到表示接下边收到的1~16为控制继电器延时打开,
字符d,接收到表示接下边收到的1~16为控制继电器延时关闭,

字符e,所有16路继电器都打开
字符f,所有16路继电器都关闭
字符g,所有16路继电器都延时打开一段时间后关闭
字符h,所有16路继电器都延时关闭一段时间后打开


```python
MSGTYPE = 0       #消息类型,
KEY_Delay = 10    #继电器延时时间,单位秒
#0表示1~16号为继电器关闭,1表示1~16号为继电器打开,2表示1~16号为继电器延时打开,3表示1~16号为继电器延时关闭

#16个按键的延时时间
delayKes = {1:-1,2:-1,3:-1,4:-1,5:-1,6:-1,7:-1,8:-1,9:-1,10:-1,11:-1,12:-1,13:-1,14:-1,15:-1,16:-1}
keyStates = {1:1,2:1,3:1,4:1,5:1,6:1,7:1,8:1,9:1,10:1,11:1,12:1,13:1,14:1,15:1,16:1} #0表示继电器关闭,1表示继电器打开

def keyChange(p):
    if MSGTYPE == 0:
        unTouchPin(p)
        keyStates[p] = 1
        delayKes[p] = -1
    elif MSGTYPE == 1:
        touchPin(p)
        keyStates[p] = 0
        delayKes[p] = -1
    elif MSGTYPE == 2:
        delayKes[p] = KEY_Delay
        unTouchPin(p)
        keyStates[p] = 1
    elif MSGTYPE == 3:
        delayKes[p] = KEY_Delay
        touchPin(p)
        keyStates[p] = 0

#巴法云回调函数,收到巴法云发来的消息后会被调用
def bfCallback(dat):
    global MSGTYPE
    print(dat)
    intdat = -1
    if is_int(dat): #收到按钮1~按钮16的点击指令,对应点击头点击一次
        intdat = int(dat)
    if intdat >=1 and intdat <= 16:   #对次编号的点击头点击一次
        keyChange(intdat)
    elif dat == 'a':           #0表示1~16号为继电器关闭
        MSGTYPE = 0
    elif dat == 'b':           #1表示1~16号为继电器打开
        MSGTYPE = 1
    elif dat == 'c':           #2表示1~16号为继电器延时打开
        MSGTYPE = 2
    elif dat == 'd':           #表示1~16号为继电器延时关闭
        MSGTYPE = 3
    elif dat == 'e':           #所有继电器打开
        setAllPinStates(0xffff)
        for i in range(1,17):
            keyStates[i] = 1
            delayKes[i] = -1
    elif dat == 'f':           #所有继电器关闭
        setAllPinStates(0x0000) 
        for i in range(1,17):
            keyStates[i] = 0
            delayKes[i] = -1
    elif dat == 'g':           #所有继电器延时打开一段时间
        setAllPinStates(0xffff)
        for i in range(1,17):
            keyStates[i] = 1
            delayKes[i] = KEY_Delay
    elif dat == 'h':           #所有继电器延时关闭一段时间
        setAllPinStates(0x0000)
        for i in range(1,17):
            keyStates[i] = 0
            delayKes[i] = KEY_Delay
    else:
        print('unknow command:%s'%(dat)) #你们发的命令,自行解析吧
```
